HTML What & Why

HTML (HyperText Markup Language) is the standard language for creating web pages and web applications. It structures your content with elements like headings, paragraphs, images, and links.


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
  • Tip: Always use semantic tags (like <header>, <main>, <footer>).
  • Resource: MDN HTML Guide

CSS Styling the Web

CSS (Cascading Style Sheets) lets you design and style your web pages. Make them colorful, responsive, and animated!


body {
  background: linear-gradient(120deg, #e0e7ff, #f9fafb);
  font-family: 'Poppins', sans-serif;
}

h1 {
  color: #2d3748;
  text-align: center;
}
  • Tip: Use Flexbox or Grid for layouts.
  • Resource: Flexbox Guide

JavaScript Make it Dynamic!

JavaScript adds interactivity and logic to your website. Handle user actions, fetch data, build games, and more!


// Show an alert on button click
document.querySelector('.btn').onclick = function() {
  alert('Welcome to my portfolio!');
};
  • Tip: Always keep your JS code organized and use console.log() for debugging.
